# Audio Library Toolkit – Silence Scanner

Ultra-fast intro and outro silence detection for large audio libraries.

Originally developed for Radio Albena to scan thousands of songs for long intros and outros before importing them into radio automation software.

## Features

- Fast intro scan
- Fast outro scan
- Multi-threaded processing
- CSV report
- Scans only intro and outro (very fast)
- Supports:

  - MP3
  - FLAC
  - WAV
  - AAC
  - M4A
  - OGG

## Why?

Many music libraries contain songs with:

- very long intros
- very long fade-outs

Those songs are not ideal for radio automation.

This tool finds them automatically.

## Requirements

- Windows
- Python 3.10+
- FFmpeg

## Usage

Run

```bash
py silence_scanner_v0.3.0.py
```

Enter your music folder.

After the scan a CSV report is created.

## Example

```
Files scanned : 6191

Long Intro : 0

Bad Outro : 4

Errors : 0

Elapsed : 00:30:52
```

## Output

The generated CSV contains only files that need attention.

Example:

```
File;Intro;Outro;Problem

Song.mp3;16.25;2.09;LONG INTRO
```

## Philosophy

This project follows one simple rule:

> **Do one thing. Do it well.**

The scanner only finds problem files.

It never modifies audio files.

## Future

Next planned tool:

- Intro / Outro Cutter

The cutter will read the CSV generated by the scanner and create trimmed copies of the affected files.

Original files will never be modified.

Typical use case

Scan your complete music library once.

Afterwards, scan only newly added tracks before importing them into your radio automation software.
